HTML5란?

HTML5는 웹페이지의 구조와 내용을 만드는 최신 웹 표준 마크업 언어입니다.

HTML5 태그 종류

  • <header> : 웹페이지나 섹션의 머리 부분을 나타내는 태그
  • <section> : 주제별 콘텐츠 영역을 나누는 태그
  • <article> : 독립적으로 의미가 있는 콘텐츠를 나타내는 태그
  • <nav> : 네비게이션 메뉴를 나타내는 태그

HTML5의 다양한 기능

  • 더 이상 영상과 음악을 플러그인 없이 바로 재생할 수 있습니다.
  • 사용자의 위치 정보를 활용할 수 있습니다.
  • 데이터를 브라우저에 저장할 수 있는 로컬 저장소 기능을 제공합니다.
  • 인터넷 연결이 없어도 실행할 수 있는 오프라인 웹앱을 지원합니다.
